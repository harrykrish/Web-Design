Steps to run the application:

1) Place the NEU_Sanskriti folder into the www directory of wamp server.
2) Open the index.php file to view the application
3) Connect to the database named sanskriti_db. Data Dump is available in sanskriti_db_dump.sql.
4) style.css contains all the required styles for the application.
5) To access the temporary accommodation form, the user needs to be signed in.
